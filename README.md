Plant Identifier
=====

A mobile app to recognize plants from pictures of flowers. This app was built using Tensorflow Image recognition as part of my software project class.


Used
====

* React Native
* Tensorflow
* Firebase
* Native Base
* Expo
* create-react-native-app
* MediaWiki
